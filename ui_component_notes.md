# UI Component Analysis - Growik Live Site

## Button Styles Observed

### Primary Button (CTA)
- **Text:** "ABOUT US", "LEARN MORE", "GET STARTED", "SEE ALL", "ASK A QUESTION"
- **Style:** Black background, white uppercase text, rounded corners
- **Hover:** Likely scale or color transition
- **Usage:** CTAs throughout sections

### Link Button (View)
- **Text:** "VIEW" (on project cards)
- **Style:** Underlined text style
- **Usage:** Project cards

### Navigation Links
- **Style:** Black pill-shaped buttons with white text
- **Items:** HOME, ABOUT US, CONTACT, CASE STUDIES
- **Hover:** Likely background color change

### Social Icons
- **Style:** Circular black buttons with white icons
- **Icons:** Twitter (X), Instagram, YouTube
- **Size:** Small, consistent sizing

## Badge Components

- **Shape:** Circular badge with rotating text
- **Text:** "SCROLL DOWN - SCROLL DOWN" (circular)
- **Icon:** Arrow pointing down in center
- **Animation:** Continuous rotation, speeds up on scroll

## Card Components

### Service Card
- **Layout:** Full-width stacking cards
- **Number:** Large "01", "02", "03", "04" (animated count-up)
- **Title:** Service name
- **Description:** Text content
- **Features:** Bulleted list
- **Background:** Video with overlay
- **Marquee:** Scrolling text at bottom
- **Animation:** Stacking with pin effect

### Project Card
- **Layout:** Grid layout (4 cards)
- **Image:** Large project thumbnail
- **Title:** Project name + year
- **Link:** "VIEW" button
- **Animation:** Fade in + slide up

### Testimonial Card
- **Rating:** "5/5", "4.8/5" in large text
- **Quote:** Testimonial text
- **Author:** Name + title
- **Style:** Clean, minimal design
- **Animation:** Horizontal carousel

### Pricing Card
- **Plan Name:** "STARTER", "GROWTH", "PREMIUM"
- **Price:** "$499 / month"
- **Description:** Plan description
- **Features:** Bulleted list with checkmarks
- **Background:** Video with overlay
- **Badge:** "7-day money-back guarantee"
- **CTA:** "GET STARTED" button

## Color Palette
- **Primary:** Orange/coral (#FF6B4A approximate)
- **Background:** Orange for hero, white/light for sections
- **Text:** Black for body, white for contrast
- **Accent:** Black buttons, white text

## Typography
- **Headings:** Bold, uppercase, large scale
- **Body:** Clean sans-serif
- **Numbers:** Large, bold for counters and stats

## Common Patterns
1. Black pill-shaped buttons for primary actions
2. Uppercase text for emphasis
3. Large bold numbers for stats/counters
4. Video backgrounds with dark overlays
5. Marquee text for dynamic movement
6. 3D rotating text for section titles
